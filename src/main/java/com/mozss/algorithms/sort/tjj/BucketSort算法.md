- 桶排序：利用函数映射关系的排序方式，非比较型排序。
### 概要
##### 性质
- 稳定
- 时间：O(n+k)
- 空间：O(n+k)
##### 算法思想
将数分布到有限数量的桶中。
### 解读
##### 简要思路
把区间划分为n个大小的区间，这样的区间称为桶。将待排序的数分布到各个桶里面，然后依次将桶里面的数列出来。
